# Kinetic - AI Personal Trainer

A modern fitness app frontend built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- **Natural Language Coaching** - Chat with your AI trainer using plain English
- **Adaptive Workouts** - AI-powered workout programs that evolve with you
- **Progress Tracking** - Comprehensive analytics and insights

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Modern React patterns

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

### Build for Production

```bash
npm run build
npm start
```

## Project Structure

```

├── app/              # Next.js App Router pages

│   ├── Footer.tsx
│   └── Container.tsx

```
