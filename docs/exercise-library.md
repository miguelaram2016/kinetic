## Equipment Master List

The following equipment types are supported in the Kinetic exercise library:

| # | Equipment | Description |
|---|-----------|-------------|
| 1 | **Barbell** | Standard Olympic barbell (45 lb/20 kg) with weight plates |
| 2 | **Dumbbell** | Adjustable dumbbells or fixed-weight dumbbells |
| 3 | **Kettlebell** | Cast iron kettlebells (various weights) |
| 4 | **Machine** | Resistance machines (e.g., leg press, chest press, lat pulldown) |
| 5 | **Cable** | Cable stations with adjustable pulleys |
| 6 | **Bodyweight** | No equipment needed—uses gravity and body weight |
| 7 | **Bench** | Flat bench, incline bench, or decline bench |
| 8 | **Pull-up Bar** | Wall-mounted or doorway pull-up bar |
| 9 | **Dip Station** | Parallel bars for dips and leg raises |
| 10 | **Treadmill** | Stationary running machine |
| 11 | **Bike** | Stationary bike (spin bike or recumbent) |
| 12 | **Rowing Machine** | Ergometer (rowing machine) |
| 13 | **Resistance Band** | Elastic bands for progressive resistance |
| 14 | **Medicine Ball** | Weighted balls for explosive movements |

## Muscle Groups

The following muscle groups are tracked in the Kinetic app:

| # | Muscle Group | Description |
|---|--------------|-------------|
| 1 | **Chest** | Pectoralis major and minor |
| 2 | **Upper Back** | Latissimus dorsi (lats), rhomboids, trapezius |
| 3 | **Lower Back** | Erector spinae (spinal erectors) |
| 4 | **Shoulders (Front)** | Anterior deltoids |
| 5 | **Shoulders (Side)** | Lateral deltoids |
| 6 | **Shoulders (Rear)** | Posterior deltoids |
| 7 | **Biceps** | Biceps brachii (long and short head) |
| 8 | **Triceps** | Triceps brachii (long, lateral, medial head) |
| 9 | **Forearms** | Forearm flexors and extensors |
| 10 | **Quadriceps** | Quadriceps femoris (rectus femoris, vastus lateralis, vastus medialis, vastus intermedius) |
| 11 | **Hamstrings** | Biceps femoris, semitendinosus, semimembranosus |
| 12 | **Glutes** | Gluteus maximus, medius, and minimus |
| 13 | **Calves** | Gastrocnemius and soleus |
| 14 | **Core** | Rectus abdominis, obliques, transverse abdominis |

## Notes for App Implementation

- **Progression Tags:** Mark foundational movements (squat, bench, deadlift, OHP, rows, pull-ups) as priority exercises for new users
- **Equipment Filters:** Allow filtering by available equipment to show only accessible exercises
- **Difficulty Scaling:** Provide regression/progression options for each exercise (e.g., push-up → weighted push-up)
- **Video References:** Each exercise should have accompanying video demonstration links
- **Form Cues:** Include 2-3 key form cues per exercise to ensure safe execution
